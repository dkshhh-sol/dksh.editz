# DKSH — Editor Portfolio

Portfolio site for DKSH, a freelance video editor & motion designer — high-retention edits, cinematic color, and motion graphics.

Built with [TanStack Start](https://tanstack.com/start) (React 19 + Vite), Tailwind CSS v4, Radix UI, and Framer Motion.

## Getting started

```bash
npm install
npm run dev
```

The dev server runs on http://localhost:8080 (falls back to the next free port).

## Scripts

| Command            | Description                          |
| ------------------ | ------------------------------------ |
| `npm run dev`      | Start the dev server                 |
| `npm run build`    | Production build (Nitro output)      |
| `npm run preview`  | Preview the production build         |
| `npm run lint`     | Run ESLint                           |
| `npm run format`   | Format with Prettier                 |

## Tech stack

- **Framework:** TanStack Start / TanStack Router
- **Build:** Vite 8
- **Styling:** Tailwind CSS v4
- **UI:** Radix UI primitives (shadcn-style components in `src/components/ui`)
- **Animation:** Framer Motion, Lenis smooth scroll
- **Data:** TanStack Query
